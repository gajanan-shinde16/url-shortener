# 🔗 URL Shortener

A scalable URL shortener built with React, Node.js, MySQL, and Redis.

## Features
- JWT Authentication (cookies-based)
- Redis caching for fast redirects
- Rate limiting to prevent abuse
- Analytics dashboard (click tracking)
- Short URL generation

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MySQL
- Cache: Redis

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
